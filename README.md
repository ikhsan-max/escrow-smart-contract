# Escrow Smart Contract

A minimal Ethereum escrow smart contract implemented using Solidity and tested with Hardhat.

## Features
- Buyer deposits ETH into escrow
- Seller receives funds after release
- Refund mechanism for buyer
- Explicit state machine to prevent invalid transitions

## Tech Stack
- Solidity ^0.8.x
- Hardhat
- Ethers.js
- Mocha / Chai
